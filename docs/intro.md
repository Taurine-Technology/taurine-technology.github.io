# Welcome to SDN Launch Control Documentation

SDN Launch Control is a comprehensive Software-Defined Networking (SDN) management platform that enables easy and automated adoption of the SDN paradigm into new or pre-existing networks.

## What is SDN Launch Control?

SDN Launch Control provides:

- **Device Management**: Automated Open vSwitch installation and SDN controller support
- **Traffic Classification**: ML-based analysis using TensorFlow/Keras
- **Network Monitoring**: Real-time device statistics and historical data visualization
- **Plugin System**: Extensible architecture for custom network functionality
- **Automation**: Ansible-based deployment and configuration
- **Notifications**: Telegram integration for alerts

- **Backend**: Django 5.0.1 REST API with WebSocket support
- **Frontend**: Next.js 15.4.2 with React 19.1.0
- **Database**: PostgreSQL 17 with TimescaleDB
- **ML**: TensorFlow 2.11.0 with Keras 2.11.0
